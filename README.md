# Laboratorio 1 
## Resumen de la Intención del Laboratorio

El objetivo de este laboratorio es practicar el flujo de trabajo básico con Git y GitHub, específicamente:
- Creación de un nuevo repositorio
- Manejo de múltiples ramas (branching)
- Trabajo concurrente en diferentes ramas
- Carga de archivos desde distintas ramas
- Comprensión de la estructura jerárquica de ramas
